聲繭/音繭 cocoonX

繭的裝置，原自於一個未來住屋的天真想像，以仿生自然繭的形態，想像人們生活居住於自然的繭屋之中，
這個想像開啟了聲音、藝術、住民、城市與環境間的想像力。
採集收錄各種屬於台南在地的文化與自然的聲響，如水流、雞鳴、市場叫賣與對話、鳥叫、蛙鳴、鐘聲等等，後製編輯後創造出台南的獨特城市音景，
這些自然與文化的聲聲響分別住進安置在海安路上散落的各個繭聲音裝置裡，依據不同的時間將播放出迥異的城市音景，提供人們聆聽。
如同建構與召喚回屬於五條港的歷史想像與海安路的生活記憶。

海安路燈柱上的小繭聲音裝置依時間序，將會陸續播放不同的音景聲音內容，大繭音景裝置則於每個整點播放音景內容，夜晚時，所有的繭裝置會散發出仿生呼吸般的光線。
